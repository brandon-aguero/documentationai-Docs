## Quick Start

Getting started is straightforward:

```javascript
const neuroglia = require('neuroglia-sdk');

neuroglia.init({
  apiKey: 'your-api-key',
  region: 'mexico'
});

// Integrate with your clinical systems
const integration = await neuroglia.integrate({
  systemType: 'hospital-emr',
  endpoint: 'https://hospital.example.com/api'
});
```
